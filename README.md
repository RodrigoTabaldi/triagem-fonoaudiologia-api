# Sistema de Triagem em Fonoaudiologia

API REST desenvolvida em C# com ASP.NET Core para gerenciamento de pacientes e realização de triagens com classificação automática de risco.

## Sobre o projeto

Este projeto foi desenvolvido para um projeto de pesquisa, com o objetivo de auxiliar no processo de triagem em fonoaudiologia.

A aplicação permite registrar pacientes, coletar respostas e processar essas informações para classificar automaticamente o nível de risco, além de armazenar o histórico das triagens.

## Funcionalidades

- Cadastro de pacientes  
- CRUD completo  
- Sistema de triagem com cálculo de risco  
- Armazenamento de dados em banco  
- Consulta de pacientes e triagens  
- Testes de endpoints com Swagger  

## Tecnologias utilizadas

- C#  
- ASP.NET Core  
- Entity Framework Core  
- SQL Server  
- Swagger  

## Estrutura do projeto

- Triagem.API → Camada de apresentação (controllers)  
- Triagem.Domain → Regras de negócio  
- Triagem.Infrastructure → Acesso a dados
